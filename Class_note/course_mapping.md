# Artful Bytes Course - Topic-Based Lesson Grouping

Based on the 28 lessons of the Artful Bytes Embedded Systems series, here are suggested groupings by related topics:

---

## Detailed Topic Grouping (10 Categories)

### **1. Project Foundation & Planning** (Lessons 1-3, 6-7)

- **Lesson 6**: Software Architecture
- **Lesson 7**: Project Structure
*Focus: Project planning, hardware design, and software organization principles.*

---

### **2. Development Environment & Build System** (Lessons 4-5)
- **Lesson 4**: IDE Setup (Code Composer Studio)
- **Lesson 5**: Makefile + Toolchain
*Focus: Setting up development tools and build automation.*

---

### **3. Professional Development Practices** (Lessons 8-11)
- **Lesson 8**: Git Version Control
- **Lesson 9**: Static Analysis (cppcheck)
- **Lesson 10**: CI/CD with GitHub Actions
- **Lesson 11**: Documentation & Code Formatting
*Focus: Professional software development workflows, quality assurance, and collaboration.*

---

### **4. Core Embedded Programming Concepts** (Lessons 14-17)
- **Lesson 14**: Assert on Microcontroller
- **Lesson 15**: Test Functions
- **Lesson 16**: Microcontroller Memory
- **Lesson 17**: Interrupts
*Focus: Fundamental embedded programming concepts, debugging techniques, and system behavior.*

---

### **5. GPIO & Hardware Abstraction** (Lessons 12-13)

---

### **6. Communication Protocols & Debugging** (Lessons 18-20)

### **7. Sensor Interfaces** (Lessons 22-23)

---

### **8. Actuator Control** (Lesson 21)
- **Lesson 21**: Motor Control with PWM
*Focus: Controlling motors and other actuators using pulse-width modulation.*

---

### **9. System Integration & Testing** (Lessons 24-27)

- **Lesson 27**: Simulation to Real-World

### **10. Reflection & Lessons Learned** (Lesson 28)
