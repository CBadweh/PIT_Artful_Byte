When adding supplementary content (diagrams, explanations, code snippets) for a lesson, **append it to the relevant section file** (`Artful_Bytes_Transcript/S0X_*.md`) under that lesson's section. Do not create standalone files in `Chat/Class_note/` or elsewhere.

**Why:** Three reasons:
1. **Single retrieval point** — one file per topic means "check S04, Lesson 008" instead of hunting across directories
2. **Co-location** — everything about a lesson (summary, terminology, techniques, diagrams) in one scroll
3. **Context efficiency** — reading one file loads everything vs. multiple files for the same lesson

Standalone files become orphans forgotten in weeks. Co-located content stays findable.

**How to apply:** When the user provides or asks for additional material for a lesson, find the matching section file and append under the correct lesson heading — before the `---` separator that starts the next lesson.
